Базовый проект приложения Be Better - эко-приложение, позволяющее стать лучше вместе с планетой. В приложении есть возможность просматривать статьи, список эко-привычек (добавлять, редактировать, удалять)
Для запуска необходимо выполнить команды:

`npm install`
`npm start`

(для запуска Android приложения используем Android Studio - Pixel 5 API 28, Pixel 3a API 34)

Экраны приложений:
Стартовый экран - Start.js
Главный экран - Main.js
Экран детальной статьи - Article.js(Список статей Articles.js)
Навигация между экранами - navigate.js
Экраны с эко-привычками (Habits.js, Habit.js)
Добавление, редактирование (AddHabit.js, EditHabit.js)

Папка с тестами: __test__
Папка с медиа для публикации : publish
